# Task Control Hub

Task Control Hub is a right-sidebar dashboard for managing all tasks across your Obsidian vault.

## Features

- Scans the entire vault for Markdown checklist tasks (`- [ ] ...`)
- Inline editing of **Status**, **Priority**, and **Due Date**
- Color-coded status (Open, In Progress, Complete, Canceled)
- Filters:
  - Status: All / Open / In Progress / Complete / Canceled
  - Priority: All / High / Medium / Low
  - Due: All / Today / This Week / Overdue
- Sort by: Created Date, Due Date, Priority, Status

## Installation (via BRAT)

1. Install the [BRAT](https://github.com/TfTHacker/obsidian42-brat) plugin.
2. In BRAT, choose **Add Beta plugin**.
3. Use the GitHub repo: `your-username/obsidian-task-control-hub`.
4. Let BRAT install it, then enable **Task Control Hub** in Community plugins.
5. Use the ribbon icon or Command Palette to open the Task Control Hub view.
